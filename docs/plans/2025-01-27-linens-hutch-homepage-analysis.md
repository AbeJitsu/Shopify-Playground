## Design System Observations

### Typography
- **Headings**: Montserrat (or similar), uppercase, wide letter-spacing (0.1-0.2em)
- **Body**: Nunito Sans, regular weight, normal case
- **Logo**: Custom script/cursive font

### Color Palette
```
Primary Dark:    #3d3d3d (charcoal) - announcement, commitment, instagram
Background:      #ffffff (white)
Text Dark:       #121212 (near-black)
Text Light:      #ffffff (white on dark)
Accent:          Muted earth tones (from product imagery)
Star Rating:     Gold/amber
Badge:           Teal/green for "NEW ARRIVAL"
```

### Spacing Rhythm
- Section padding: ~60-80px vertical
- Content max-width: ~1200px
- Grid gaps: ~20-30px
- Card padding: ~16-20px

### Recurring Components

1. **Product Card**
   - Square image (1:1 ratio)
   - Optional badge overlay
   - Product title (2-3 lines, truncated)
   - Star rating + review count
   - "From $XXX.00" price
   - Color swatch dots (max 6 visible)

2. **Feature Block**
   - 50/50 two-column layout
   - Alternates image left/right
   - Large stacked headline
   - Short body paragraph
   - Dark CTA button

3. **Product Carousel**
   - 4 products visible (desktop)
   - Pagination counter (1/23)
   - Prev/next arrows
   - "VIEW ALL" link option

4. **Section Header**
   - Centered uppercase title
   - Optional centered subtitle
   - Generous whitespace above/below

## Improvement Opportunities

Areas where we can enhance while rebuilding:

1. **Performance**
   - Lazy load carousel images
   - Use `srcset` for responsive images
   - Preload hero image

2. **Accessibility**
   - ARIA labels on carousels
   - Keyboard navigation for sliders
   - Focus states on interactive elements
   - Screen reader text for star ratings

3. **Mobile Experience**
   - Touch-friendly carousel swipe
   - Collapsible mega-menu
   - Stack feature blocks vertically

4. **Schema Flexibility**
   - Make badge text customizable
   - Allow variable number of trust badges
   - Configurable carousel product count
